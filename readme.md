
# MDM Connect API

API di connessione a Dateq MDM-D8 completo di esempio in Flask funzionante di pagina web di controllo


## Deployment

Per avviare il progetto avvia

```bash
  python app.py
```
E punta il browser a 127.0.0.1:5000
